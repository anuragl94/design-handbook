Fonts

Our font should be consistent across the website. Recommended fonts are "Source Sans Pro", "Open Sans", "Nunito", "Lato", "Roboto".
We'll use "Source Sans Pro".

We should also specify widely available fonts such as "Arial" or "sans-serif" as fallbacks in case of poor networks or unavailability of hosting service that offers the font.

To represent visual hierarchy in text, we should use different font sizes.
A good way to handle different font sizes is to set a global font size on body (16px) and use the unit em to choose a relative font size.

Layout

Your page should be accessible on every screen size.

On small screens, occupy all of the width.

On larger screens, set a max-width on your content so that it doesn't flow freely into infinite width.
This makes it easier to test your UI by limiting the "states" your views can take.
